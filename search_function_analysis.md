# Search Function 输入和输出分析

## 概述
该代码库中的"search"功能是用于蛋白质-蛋白质相互作用（PPI）中的**配对匹配预测**，通过 `--search` 参数控制。当启用时，模型会预测两个蛋白质伙伴之间的匹配关系。

## 输入 (Inputs)

### 1. 模型输入
- **P1, P2**: 两个蛋白质的数据字典，包含：
  - `xyz`: 蛋白质表面点的3D坐标
  - `normals`: 表面法向量
  - `batch`: 批次信息
  - `atoms`: 原子信息
  - `atomtypes`: 原子类型
  - `triangles`: 网格三角形（如果使用mesh）

### 2. 模型架构输入
- **features**: 从原子信息提取的特征向量
- **embedding_layer**: 使用的嵌入层类型（dMaSIF, DGCNN, PointNet++）
- **n_layers**: 网络层数
- **radius**: 邻域半径

### 3. 配置参数
- **args.search**: 布尔值，启用搜索模式
- **args.random_rotation**: 是否使用随机旋转
- **args.use_mesh**: 是否使用网格信息

## 输出 (Outputs)

### 1. 嵌入向量
- **P1["embedding_1"]**: 蛋白质1的第一组嵌入向量
- **P1["embedding_2"]**: 蛋白质1的第二组嵌入向量（仅在search模式下）
- **P2["embedding_1"]**: 蛋白质2的第一组嵌入向量  
- **P2["embedding_2"]**: 蛋白质2的第二组嵌入向量（仅在search模式下）

### 2. 接触界面标签
- **P1["labels"]**: 蛋白质1表面点的二进制标签（1=接触界面，0=非接触界面）
- **P2["labels"]**: 蛋白质2表面点的二进制标签

### 3. 损失计算输出
- **loss**: 二进制交叉熵损失值
- **preds_concat**: 连接的预测分数
- **labels_concat**: 连接的真实标签

### 4. 匹配预测
- **pos_preds**: 正样本（真实接触点对）的相似度分数
- **neg_preds**: 负样本（随机采样点对）的相似度分数
- **pos_desc_dists**: 正样本描述符之间的点积距离
- **pos_xyz_dists**: 正样本3D空间距离

## 核心工作流程

1. **双向嵌入**: 为每个蛋白质生成两组不同的嵌入向量（embedding_1和embedding_2）
2. **接触界面检测**: 基于3D距离（<1.0Å）识别蛋白质间的接触界面
3. **匹配评分**: 计算接触点对之间的描述符相似度
4. **对称性处理**: 同时计算P1→P2和P2→P1的匹配分数
5. **损失计算**: 使用正负样本对比学习训练模型

## 应用场景
- 蛋白质对接预测
- 结合位点识别  
- 蛋白质相互作用界面分析
- 分子识别和匹配
